# 屬性說明

屬性名稱                 |屬性說明
------------------------|-------------------------------
data-gt-target          |要前往的目標元素，必須使用ID
data-gt-duration        |持續時間，單位為毫秒
data-gt-offset          |位移，往上位移的位置
